# Beta 0.9.6-beta.1

* Add multiplayer lobby navigation, LAN world discovery, and Terracotta-backed lobby creation and joining.
* Add server installation targets for supported modpacks, with safer loader provisioning and deployment rollback.
* Improve Minecraft download scheduling, mirror fallback, integrity verification, repair coverage, and progress reporting.
* Add strict Java compatibility selection and support split native directories required by newer Minecraft versions.
* Improve resource browsing, local content management, download settings, diagnostics, backups, and launcher reliability.
* Handle restricted CurseForge downloads with CDN fallbacks and improved concurrency control.
* Remove update-manifest signing while retaining HTTPS source restrictions and executable size and SHA-256 validation. Older signed-manifest builds may require a manual update to this version.
