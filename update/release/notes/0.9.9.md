# 正式版 0.9.9

* 新增自定义文件下载功能，可输入 HTTP 或 HTTPS 文件地址并选择本地保存位置。
* 升级游戏下载引擎，为大型 Minecraft 文件提供自适应分段下载，并改进断点续传、失败重试、完整性校验和同源并发控制。
* 优化下载速度统计、限速和并发调度，提高资源依赖项与整合包文件的批量下载效率。
* 新增启动后自动加入服务器设置，支持全局默认配置和实例独立配置，并根据游戏版本使用兼容的启动参数。
* 改进 Java 运行时导入与发现，手动导入的有效路径可在刷新和重启后保留，失效路径会在重新扫描时清理。
* 改进本地整合包识别，扩展名为 `.zip` 的 Modrinth 整合包也可根据包内清单正常导入。
* 在资源详情页新增相关网站入口，并完善资源关联信息的获取与展示。
* 完善关于页面中的引用项目与许可信息展示，并补充多语言文案。
* 修复亚克力背景不透明度为 `100%` 时数值换行的问题。
